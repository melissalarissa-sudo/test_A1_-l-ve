H muet vs H aspiré — par cœur !

H MUET → l’
l’homme • l’hôtel • l’heure
l’histoire • l’hôpital • l’hiver
l’habitude • l’huile
→ élision obligatoire
H ASPIRÉ → le / la
le hibou • le hall • le haricot
le hasard • la honte • la harpe
le homard • le hockey
→ pas d’élision
Piège : le héros (H aspiré) MAIS l’héroïne (H muet) — même famille !
